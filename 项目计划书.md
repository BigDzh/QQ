# 项目全生命周期管理系统 - 项目计划书

## 一、项目概述

### 1.1 系统定位

模块组件全生命周期管理系统是一个用于管理项目中模块和组件从创建到退役全过程的信息化系统。系统采用三级管理架构，支持项目、模块种类、模块组件的层级管理。

### 1.2 技术架构

- **前端框架**: React 18 + TypeScript
- **路由**: React Router v6
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **数据存储**:
  - localStorage 本地持久化存储（项目数据）
  - **IndexedDB 本地数据库（大文件存储）**
- **状态管理**: React Context API
- **打包工具**: Vite + Electron（桌面应用）
- **Web应用**: 支持直接浏览器运行
- **辅助工具**:
  - recharts（数据可视化）
  - xlsx（Excel 导入导出）
  - MD5 加密工具
  - Tesseract.js（OCR文字识别）

### 1.3 系统特性

- ✅ 用户权限系统（JWT Token）
- ✅ 角色权限管理（4种角色）
- ✅ **登录功能**（登录页面、状态持久化、退出登录）
- ✅ 数据备份与恢复
- ✅ 操作审计日志
- ✅ 错误边界处理
- ✅ Toast 提示组件
- ✅ **文档管理增强**（按阶段分类、下载/预览/发送）
- ✅ **软件管理增强**（MD5校验、版本管理）
- ✅ **设计文件管理**（自动生成、CSV导出）
- ✅ **Web应用打包**（可直接浏览器运行）
- ✅ **数据库管理**（IndexedDB本地数据库）
- ✅ **全局搜索功能**（Ctrl+K快捷键、模糊搜索、搜索历史）
- ✅ **任务到期提醒**（倒计时显示、紧急程度标识）
- ✅ **批量操作功能**（多选、批量删除、批量导出）
- ✅ **操作撤销功能**（10秒撤销窗口、倒计时提示）
- ✅ **数据导入向导**（Excel解析、导入预览、模板下载）
- ✅ **项目进度看板**（甘特图、时间线展示）
- ✅ **数据备份提醒**（定期提醒、一键备份）
- ✅ **快速入门指南**（新手引导、常见问题FAQ）
- ✅ **打印优化**（打印预览、打印样式优化）

## 二、系统架构

### 2.1 三级管理结构

```
┌─────────────────────────────────────────────────────────┐
│  第一级：项目管理                                        │
│  - 项目列表                                             │
│  - 项目详情（概览、统计、文档、软件、日志）                    │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│  第二级：模块种类管理                                    │
│  - 种类列表（按字母/类型分类）                           │
│  - 种类下模块管理                                       │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│  第三级：模块及组件管理                                   │
│  - 模块详情（证书、日志、状态变更）                       │
│  - 组件列表                                             │
│  - 组件详情（证书、日志、状态变更）                       │
└─────────────────────────────────────────────────────────┘
```

### 2.2 页面路由

| 路由路径                               | 页面    | 功能            |
| ---------------------------------- | ----- | ------------- |
| `/login`                           | 登录    | 用户登录          |
| `/projects`                        | 项目列表  | 显示所有项目        |
| `/projects/:id`                    | 项目详情  | 项目概览、统计、文档管理  |
| `/projects/:id/category/:category` | 模块种类  | 种类下的模块列表      |
| `/projects/:id/files`              | 文件管理  | 项目文档和软件管理     |
| `/modules/:id`                     | 模块详情  | 模块信息、组件管理     |
| `/components/:id`                  | 组件详情  | 组件信息、证书、日志    |
| `/tasks`                           | 任务管理  | 任务、故障组件、未完成文档 |
| `/borrow`                          | 借用系统  | 模块组件借用管理      |
| `/tools`                           | 工具技能  | MD5生成、OCR识别   |
| `/users`                           | 用户管理  | 用户CRUD、权限分配   |
| `/backups`                         | 备份管理  | 数据备份、恢复、导出导入  |
| `/audit-logs`                      | 审计日志  | 操作记录查询、导出     |
| `/database`                        | 数据库管理 | IndexedDB文件管理 |

## 五、数据模型

### 5.1 用户 (User)

```typescript
{
  id: string
  username: string
  password: string
  name: string
  email: string
  role: 'admin' | 'manager' | 'engineer' | 'viewer'
  avatar?: string
  createdAt: string
  lastLogin?: string
}
```

### 5.2 角色权限 (Role)

```typescript
{
  id: string
  name: string
  description: string
  permissions: Permission[]
}
```

权限类型：

- 项目：project:read, project:write, project:delete
- 模块：module:read, module:write, module:delete
- 组件：component:read, component:write, component:delete
- 用户：user:read, user:write, user:delete
- 设计文件：designFile:read, designFile:write
- 文档：document:read, document:write
- 软件：software:read, software:write
- 任务：task:read, task:write

### 5.3 项目 (Project)

```typescript
{
  id: string
  name: string
  projectNumber: string
  stage: 'C阶段' | 'S阶段' | 'D阶段' | 'P阶段'
  version: string
  categories: string[]
  modules: Module[]
  documents: Document[]
  software: Software[]
  designFiles: DesignFile[]
  logs: ProjectLog[]
  image?: string
  createdAt?: string
}
```

### 5.4 模块 (Module)

```typescript
{
  id: string
  projectId: string
  productionOrderNumber: string
  moduleNumber: string
  moduleName: string
  category: string
  holder?: string
  status: ModuleStatus
  stage: string
  version: string
  productionNumber?: string
  components: Component[]
  logs: ModuleLog[]
  statusChanges: StatusChange[]
  designFiles?: string[]  // 关联的设计文件ID
}
```

### 5.5 组件 (Component)

```typescript
{
  id: string
  moduleId: string
  componentNumber: string
  componentName: string
  productionOrderNumber: string
  repairOrderNumber?: string
  protectionOrderNumber?: string
  holder?: string
  stage?: string
  version?: string
  status: ComponentStatus
  logs: ComponentLog[]
  certificates: Certificates
  statusChanges: StatusChange[]
  designFiles?: string[]  // 关联的设计文件ID
}
```

### 5.6 设计文件 (DesignFile)

```typescript
{
  id: string
  name: string
  type: '配套表' | '装配图'
  format: 'Excel' | 'AutoCAD' | 'PDF'
  category?: string        // 模块种类
  moduleId?: string      // 关联的模块ID
  componentId?: string   // 关联的组件ID
  stage?: ProjectStage   // 所属阶段
  version?: string       // 版本号（v1, v2, v3...）
  isAutoGenerated: boolean
  uploadDate: string
  fileUrl?: string
  fileName?: string
  fileData?: string
  fileSize?: number
  fileType?: string
  data?: string         // JSON格式的装配图/配套表数据
}
```

### 5.6 审计日志 (AuditLog)

```typescript
{
  id: string
  timestamp: string
  userId: string
  username: string
  action: AuditAction
  level: 'INFO' | 'WARNING' | 'CRITICAL'
  resourceType: string
  resourceId?: string
  resourceName?: string
  details?: string
  ipAddress?: string
}
```

### 5.7 状态类型

- **模块状态**: 未投产 | 投产中 | 正常 | 维修中 | 三防中 | 测试中 | 仿真中 | 故障
- **组件状态**: 未投产 | 投产中 | 正常 | 维修中 | 三防中 | 测试中 | 仿真中 | 故障
- **文档状态**: 未完成 | 已完成
- **软件状态**: 未完成 | 已完成
- **任务状态**: 进行中 | 已完成 | 已过期
