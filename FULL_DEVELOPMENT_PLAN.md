### 4.4 核心数据模型

```
Project (项目)
├── id: string (主键)
├── name: string
├── description: string
├── status: 'active' | 'archived' | 'deleted'
├── category: string
├── createdAt: Date
├── updatedAt: Date
└── modules: Module[]

Module (模块)
├── id: string (主键)
├── projectId: string (外键)
├── name: string
├── type: string
├── status: string
├── order: number
└── components: Component[]

Component (组件)
├── id: string (主键)
├── moduleId: string (外键)
├── name: string
├── version: string
├── description: string
└── dependencies: string[]
```
